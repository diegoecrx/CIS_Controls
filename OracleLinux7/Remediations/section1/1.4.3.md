# 1.4.3 Ensure core dump backtraces are disabled

## Compliance Requirement

**Command to Check:**
```bash

```

**Expected Output:**
```
(?i)^\h*ProcessSizeMax\h*=\h*0\b
```

## Solution

Create or edit the file /etc/systemd/coredump.conf and edit or add the following line:

ProcessSizeMax=0
