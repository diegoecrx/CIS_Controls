# 5.2.2.2 Ensure audit logs are not automatically deleted

## Compliance Requirement

**Command to Check:**
```bash

```

**Expected Output:**
```
^[\s]*max_log_file_action[\s]*=[\s]*(?i)keep_logs(?-i)[\s]*$
```

## Solution

Set the following parameter in /etc/audit/auditd.conf:

max_log_file_action = keep_logs
