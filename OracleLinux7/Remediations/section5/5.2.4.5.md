# 5.2.4.5 Ensure audit configuration files are 640 or more restrictive

## Compliance Requirement

**Command to Check:**
```bash
/bin/find /etc/audit/ -type f \( -name '*.conf' -o -name '*.rules' \) -exec /bin/stat -Lc "%n %a" {} + | /bin/grep -Pv -- '^\h*\H+\h*([0,2,4,6][0,4]0)\h*$' | /bin/awk '{print} END { if(NR==0) print "pass" ; else print "fail"}'
```

**Expected Output:**
```
^pass$
```

**Timeout:** 7200 seconds

## Solution

Run the following command to remove more permissive mode than 0640 from the audit configuration files:

# find /etc/audit/ -type f ( -name '*.conf' -o -name '*.rules' ) -exec chmod u-x,g-wx,o-rwx {} +
