# 5.2.4.7 Ensure audit configuration files belong to group root

## Compliance Requirement

**Command to Check:**
```bash
/bin/find /etc/audit/ -type f \( -name '*.conf' -o -name '*.rules' \) ! -group root | /bin/awk '{print} END { if(NR==0) print "pass" ; else print "fail"}'
```

**Expected Output:**
```
^pass$
```

**Timeout:** 7200 seconds

## Solution

Run the following command to change group to root :

# find /etc/audit/ -type f ( -name '*.conf' -o -name '*.rules' ) ! -group root -exec chgrp root {} +
