# 5.2.4.6 Ensure audit configuration files are owned by root

## Compliance Requirement

**Command to Check:**
```bash
/bin/find /etc/audit/ -type f \( -name '*.conf' -o -name '*.rules' \) ! -user root | /bin/awk '{print} END { if(NR==0) print "pass" ; else print "fail"}'
```

**Expected Output:**
```
^pass$
```

**Timeout:** 7200 seconds

## Solution

Run the following command to change ownership to root user:

# find /etc/audit/ -type f ( -name '*.conf' -o -name '*.rules' ) ! -user root -exec chown root {} +
