# 4.2.12 Ensure sshd KexAlgorithms is configured

## Compliance Requirement

**Command to Check:**
```bash
multiple line script
```

**Expected Output:**
```
^Pass$
```

## Solution

Edit /etc/ssh/sshd_config and add/modify the KexAlgorithms line to contain a comma separated list of the site approved, supported "strong" KexAlgorithms:

Example:

KexAlgorithms curve25519-sha256,curve25519-sha256@libssh.org,ecdh-sha2-nistp256,ecdh-sha2-nistp384,ecdh-sha2-nistp521,diffie-hellman-group-exchange-sha256

Note: First occurrence of an option takes precedence.
