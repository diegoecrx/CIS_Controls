# 3.4.2.4 Ensure network interfaces are assigned to appropriate zone

## Compliance Requirement

**Command to Check:**
```bash
/bin/find /sys/class/net/* -maxdepth 1 | /bin/awk -F"/" '{print $NF}' | while read -r netint; do [ "$netint" != "lo" ] && /bin/firewall-cmd --get-active-zones | /bin/grep -B1 $netint; done
```

**Expected Output:**
```
^Manual Review Required$
```

**Timeout:** 7200 seconds

## Solution

Run the following command to assign an interface to the approprate zone.

# firewall-cmd --zone= --change-interface=

Example:

# firewall-cmd --zone=customezone --change-interface=eth0

Impact:

Changing firewall settings while connected over network can result in being locked out of the system.
