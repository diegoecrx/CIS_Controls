# 2.2.2 Ensure avahi daemon services are not in use (Automated)

## Remediation

Run the following commands to stop avahi-daemon.socket and avahi-daemon.service,
and remove the avahi package:
# systemctl stop avahi-daemon.socket avahi-daemon.service
# yum remove avahi
-OR-
-IF- the avahi package is required as a dependency:
Run the following commands to stop and mask the avahi-daemon.socket and avahi-
daemon.service:
# systemctl stop avahi-daemon.socket avahi-daemon.service
# systemctl mask avahi-daemon.socket avahi-daemon.service
